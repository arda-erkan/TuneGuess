Settings

YouTube API Key is required for Custom Playlist mode.
It is used to find audio for Spotify songs, and to load YouTube playlists directly.

Get a free key at console.cloud.google.com — create a project, enable YouTube Data API v3, then create an API key.

Spotify playlists are supported without any login or API key. Just paste a public Spotify playlist URL — song names are fetched automatically, audio comes from YouTube.

YouTube playlists also work — paste any YouTube playlist URL.
